---
title: "Using the AI Resume Assistant"
description: "Learn how the AI Resume Assistant replaced the standalone agent workspace and now edits resumes from the builder."
---

The standalone AI Agent workspace has been replaced by the builder-integrated AI Resume Assistant. Open a resume in the builder and click the assistant button in the builder dock to chat with the assistant.

<Info>
  Assistant edits apply to the active resume directly. Use the builder dock's undo and redo buttons to recover from
  assistant edits, the same way you recover from manual field edits.
</Info>

## Before you start

You need at least one AI provider that is tested and enabled in **Dashboard -> Settings -> Integrations**. For setup, see [Using Artificial Intelligence](/guides/using-ai).

If you self-host Reactive Resume, the assistant also requires the server-side AI configuration described in [Self-hosting with Docker](/self-hosting/docker).

## Open the assistant

From the dashboard, open the resume you want to edit. Click the assistant button in the builder dock.

## Ask for resume changes

The assistant works best with concrete instructions:

- "Tailor this resume to this job description: `https://example.com/job`"
- "Find weak bullets and rewrite them with stronger outcomes."
- "Compare this draft against a product manager role and update the keywords."
- "Ask me before changing anything that looks uncertain."

## Review patches

When the assistant edits the resume, the patch is applied immediately to the active resume. The chat shows a **Resume updated** marker.

Use the builder dock's undo and redo buttons if you want to roll back or restore the change.

<Warning>
  AI-generated changes can still be inaccurate. Review the resume in the builder before exporting or sharing it.
</Warning>

## Answer assistant questions

If the assistant needs a decision, it may show a question card with recommended answers. Click the answer you want to send it back to the assistant.

This is useful when the assistant finds ambiguous instructions, missing job context, or a change that depends on your preference.

## Use resume analysis and imports

The builder assistant is one AI workflow. Reactive Resume also keeps:

- **Resume Analysis** in the builder right sidebar for scoring, strengths, and prioritized suggestions.
- **PDF and Microsoft Word imports** from the dashboard import dialog for turning unstructured resume files into editable builder data.

For details, see [Using AI in the builder](/guides/using-ai-in-the-builder) and [Importing resumes](/guides/importing-resumes).
